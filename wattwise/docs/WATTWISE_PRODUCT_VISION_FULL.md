# WattWise — Product Vision

## Overview

WattWise is a premium AI-powered electrician exam preparation application designed to help users successfully pass apprentice and master electrician licensing exams through structured learning, adaptive practice, and real-time AI guidance.

Unlike generic study tools, WattWise is purpose-built for the electrical trade. It combines jurisdiction-specific exam preparation, NEC (National Electrical Code) comprehension, structured modules, intelligent quizzes, and an integrated AI tutor into a calm, focused, and highly usable mobile experience.

WattWise is not designed to overwhelm the user with features. It is designed to guide the user with clarity.

---

## Vision Statement

WattWise will become the most trusted AI-powered study companion for electrician licensing in the United States.

It will be recognized for:
- accurate and reliable state-specific exam preparation
- deep and practical understanding of NEC code
- structured, easy-to-follow learning paths
- a calm, distraction-free, premium user experience

---

## Mission

To help electricians study smarter, understand deeper, and pass faster by combining structured learning systems with AI-powered guidance in a focused and intuitive environment.

---

## Core Product Scope (Locked)

WattWise is intentionally focused to maximize depth and quality.

### Included

- Apprentice electrician exam preparation
- Master electrician exam preparation
- State/jurisdiction-specific learning paths
- NEC code lookup and explanation tools
- AI-powered tutoring
- Structured modules, lessons, and quizzes
- Progress tracking and readiness indicators

### Explicitly Excluded (for now)

- HVAC or other trade expansion
- generic multi-industry learning platform features
- social-first or community-driven learning features
- unrelated certification systems

This constraint ensures WattWise remains precise, useful, and high-quality.

## Core Problem

Electricians preparing for licensing exams face a fragmented, inefficient, and often frustrating study process.

### Market Gaps

- Study tools are not tailored to specific state exams
- NEC code is difficult to search, interpret, and apply
- Platforms focus on memorization instead of understanding
- Users lack guidance on what to study next
- Progress tracking is unclear or misleading
- Many tools are visually cluttered or outdated

### User Pain Points

Users commonly experience:

- confusion about where to begin
- frustration with unclear explanations
- difficulty understanding NEC language
- lack of confidence in their readiness
- inconsistent study habits due to poor UX

## Product Philosophy

WattWise follows the TEHSO philosophy:

“The interface should disappear. The learning should remain.”

This means:

- minimal UI complexity
- calm visual design
- predictable structure
- no unnecessary features
- emphasis on clarity over stimulation

The product must reduce cognitive load, not increase it.

---

## TEHSO Alignment

WattWise must adhere strictly to TEHSO design principles:

- single accent color (TEHSO Blue #2E53FF)
- white-dominant interface
- Inter typeface exclusively
- 8px spacing system
- minimal UI elements
- line icons only
- calm, human-centered language

Consistency across TEHSO products is mandatory.
